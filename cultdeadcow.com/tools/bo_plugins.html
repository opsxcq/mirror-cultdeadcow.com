Back Orifice Unified Tool Transport plugins
This page links to all of the currently released BUTTplug plugins for Back Orifice. The Cult of the Dead Cow was not involved in the creation of these plugins, and cannot guarantee that they work as advertised. Caveat hackor.

If you have a BUTTplug that you would like listed on this page, e-mail the file, or a link to it, to bo@cultdeadcow.com.

For more information on how to write BUTTplugs, see the documentation included with the Back Orifice distribution file.
